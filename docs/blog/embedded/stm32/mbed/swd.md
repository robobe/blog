# How to connect swd to nucleo board

![alt text](image.png)

| swd  | nucleo f446RE  |
|---|---|
| **1** 3.3 v (note i connect 3.3 from JP1) |    |
| **2** SWD clock  | CN7 15  |
| **3** GND  | CN7 19  |
| **4** SWD I/O  | CN7 13  |
| **5** Reset  | CN7 14  |
| **6** Serial wire output  | CN10 31  |
